Project Idea

A Project idea I have in mind is to create a simple mobile phone app. I am passionate about those simple games you find on the app store/google play store as they are fun to play and is very user friendly. I would like to create a simple mobile phone game in the future, similar to ‘Flappy Bird’ which had millions of downloads in a short amount of time. With my project of a mobile phone game, I would avoid plagiarism of another game’s graphics, which is what ‘Flappy Bird’ did with Nintendo’s Mario series.

This project would be interesting because even if a game looks simple, it doesn’t mean that it was easy to make. The game “Flappy Bird” was apparently only created within a couple of a days, but the design was plagiarised from another game. The chances of new mobile games striking high is very slim and even ‘Flappy Bird’ took 7 months to become popular and have millions of downloads. ‘Flappy Bird’ made about $50,000 per day of advertising income which is a lot for a single developer. Although ‘Flappy Bird’ made about $50,000 a day, that’s not even much compared to a Supercells ‘Clash of Clans’, another mobile game that has had millions of downloads. Although both of these games were free, ‘Clash of Clans’ had in game purchases and about 150 developers. Even after 7 years after it’s release, the revenue made by this game daily still exceeds $1,000,000.

I have many ideas for mobile phone games but would like to start with something very basic first for my first phone mobile phone game, which could be something like a puzzle game, word game or something educational for little kids. The game would be aimed to be very user friendly as there are many apps on the Store that have received lots of complaints for being difficult to use. The first idea I have in mind is a simple chat bot, as we are currently creating one in my programming class. We are currently creating a chat bot using Eclipse, which mainly uses the Java language. With a chat bot, I would have multiple of mock conversations with different people to prepare the bot for different responses. As the chatterbot we are making in class only contains words, I would like my chatterbot app to have a character design, a voice and some movement. The second idea I have in mind is a word search, but I doubt that it would be popular as there are many word search apps already on the Play/App store. Although there are many word search apps out there, I would like to create one that’s for children, which could have words categorised into animals or numbers. The third idea I have in mind is a game similar to ‘Super Mario Bro’s, which is a 2d game, side-scrolling game. Although there was only simple movement in the game such as back, forward and jump, the game was commended for it’s precise controls, where you could control how fast the character would run or how high it can jump. With my ideas, as a project, it would be more ideal for it to be made by scratch, even if it wouldn’t be as good.

Mobile phones and a desktop computer would be required to build a mobile phone game. The app could be made by scratch with knowledge on programming. There are also many apps out there that assist with app making and some even do the programming part for you.  Some languages that’ll be useful to learn would be Python, Javascript, Java, and C++.
If the project is successful, it’ll be on the posted on the Google Play Store or Apple’s App Store. One of the drawbacks could be that if the project idea went really well, there would be a lot of clones stealing your ideas and potentially making a better phone game.
